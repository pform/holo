# Holograph — Stealth Spatial Tech Gateway

A highly classified, high-fidelity experience for **Holograph**, a stealth-mode spatial technology enterprise. Made with an elegant layout focusing on deep-black aesthetics, live coordinates tracking, and an interactive D3.js physics-driven force gravity network.

---

## Technical Architecture

- **Visual Infrastructure**: Full `#000000` pitch black layout optimized with responsive typography, iridescent hover rings, and a glassmorphism terminal panel.
- **Engine Core**: D3.js force simulation modeling localized physical vectors, real-time wave oscillations, and ambient coordinate matrices.
- **Gatekeeper Subsystem**: Passcode signature validation (e.g., `REALITY`, `CERN`, `QUANTUM`, `EVU`).
- **Communication Shield**: Cryptographic verification puzzle gatekeeper restricting automated bots from harvesting the raw target mailbox address (`info@holograph.cc`).

## Deployment (Vercel Free Tier)

1. **Upload Codebase**: Push the codebase into any public or private GitHub repository.
2. **Import Project**: Log in to [Vercel](https://vercel.com) and import the repository.
3. **Deploy & Bind Domain**: Add your custom domain (e.g., `holograph.cc`) in Vercel **Settings** -> **Domains**. No environment variables are necessary!
